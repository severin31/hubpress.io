Easy start with HubPress

Step by step installation

To install HubPress, just follow this little recipe:

  1. Fork the hubpress.io repository

  2. Rename your repository to [yourgithubusername].github.io

  3. Set values in hubpress/config.json (optional since version 0.8)

    "meta": {
    "username": "[yourgithubusername]",
    "repositoryName": "[yourgithubusername].github.io",
    "branch": "master"
}

  4. Go to https://[yourgithubusername].github.io/hubpress

  5. Login and…​ start to write ^^

Someone made a video, but i don’t think you’ll need it…​

Choose a theme

The default theme is casper…​ Let’s try another one. uno and saga themes are known for having a better code block support, so they seem to be a good choice for me. In order to do that, two possibilies: modify the hubpress/config.json file, or click the "settings" menu entry. Be patient, refreshment sometimes takes several minutes.
