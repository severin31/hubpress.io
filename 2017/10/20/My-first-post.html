How I installed Hubpress

  1. Fork the hubpress.io repository

  2. Rename your repository to kskarthik.github.io

  3. Set values in hubpress/config.json like this :

{
  "meta": {
    "username": "severin31",
    "repositoryName": "severin31.github.io",
    "branch": "master"
  }
}

  1. Go to https://severin31.github.io/hubpress

  2. Login and write
